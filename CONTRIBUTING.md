# Contributing to OpenClaw-Team

Thank you for your interest in contributing!

## How to Contribute

1. **Fork** the repository
2. **Create a feature branch**: `git checkout -b feature/my-feature`
3. **Make your changes** and commit them
4. **Push to your fork**: `git push origin feature/my-feature`
5. **Open a Pull Request**

## Guidelines

- Follow the existing code style
- Write clear, descriptive commit messages
- Include tests for new functionality
- Keep PRs focused on a single feature or fix
- Run `pnpm build && pnpm check` before submitting

## Reporting Issues

- Search existing issues before creating a new one
- Include code snippets, error messages, and environment details
- For security issues, please report privately instead of via public issues
